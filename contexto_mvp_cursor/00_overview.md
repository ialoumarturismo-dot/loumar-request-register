# Contexto Geral do Projeto

Este projeto tem como objetivo criar um MVP funcional em um único dia para centralizar demandas do setor de Tecnologia, IA e Automação.

O problema atual é a pulverização de demandas em múltiplos canais (mensagens privadas, grupos, conversas presenciais), o que gera perda de contexto, retrabalho e ausência de priorização.

A solução proposta é uma aplicação web simples, com:
- Formulário público para registro de demandas
- Painel administrativo interno para triagem e acompanhamento
- Upload de evidências (prints)
- Estrutura preparada para futuras automações e uso de IA

O foco é pragmatismo, velocidade de entrega e clareza operacional.
